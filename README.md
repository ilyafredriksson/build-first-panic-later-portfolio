Visit the live portfolio: [deploy-and-pray.vercel.app](https://deploy-and-pray.vercel.app)

## 🛠 Tech Stack

- **Framework:** Next.js 16 with App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Backend:** Node.js (for projects)
- **Database:** Convex (for realtime features)
- **Authentication:** JWT, Convex Auth
- **Deployment:** Vercel

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- pnpm (or npm/yarn)

### Installation

\\\ash
# Clone the repository
git clone https://github.com/ilyafredriksson/build-first-panic-later-portfolio.git

# Navigate to project
cd build-first-panic-later-portfolio

# Install dependencies
pnpm install

# Run development server
pnpm run dev
\\\

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build for Production

\\\ash
pnpm run build
pnpm run start
\\\
